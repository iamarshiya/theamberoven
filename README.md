### ğŸ—ï¸ Technical Foundation

This project is built on a modern, performance-first stack.

* **Runtime:** [React.js](https://react.dev/) (Vite for Hot Module Replacement).

* **State Management:** React Context API for lightweight, efficient cart logic.
